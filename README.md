# busmonitor

> 日本語のREADMEはこちらです: [README.ja.md](README.ja.md)

Open data utilization, Sabae Tsutsujibasu bus monitor

## Demo
http://codeforfukui.github.io/busmonitor/

## Features
- Displays real-time location and status of buses
- Integrates bus route and stop information
- Visualizes passenger count and wheelchair usage data

## Data / API
This project utilizes open bus data from the following sources:
- SPARQL endpoint for Fukui Open Data Portal (https://sparql.odp.jig.jp/data/sparql)
- SPARQL endpoints for other cities (Osaka, Kyoto, Kobe)
