# 鯖江つつじバスモニター
鯖江市の公共交通IoT（Internet of Things）によるリアルタイムオープンデータを利用して、バスの運行状況をウェブ上で確認できるサービスです。

## デモ
http://codeforfukui.github.io/busmonitor/

## 機能
- 鯖江市内のバス停・バス運行情報の表示
- ユーザーの現在位置の表示
- バスの位置、速度、遅延状況などの確認

## 必要環境
特に必要な環境はありません。ウェブブラウザで利用できます。

## 使い方
特別な操作は不要で、ウェブサイトにアクセスするだけで利用できます。

## データ・API
本プロジェクトでは、以下のオープンデータを利用しています:
- 鯖江市オープンデータポータル
- 大阪市、京都市、神戸市のSPARQLエンドポイント
